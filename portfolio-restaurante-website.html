Restaurante Website

O projeto foi crado para um cliente proprietário de um Restaurante italiano.

O objetivo: ganhar mais visibilidade para o restaurante, criando facilidade para mostrar o cardápio de pizza aos usuários.

Como o cliente não está muito à vontade com as tecnologias, foi desenhado para ele uma interface de administração específica, para que tenha a possibilidade de atuaizar facilmente o menu do restaurante.

Este projeto é simples mas eficiente para o seu objetivo e proporcionou total satisfação ao cliente.

Tecnologias usadas na construção: HTML, CSS, DJANGO, PYTHON, HEROKU
